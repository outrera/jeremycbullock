Software We Use

We use a variety of software in the Digital Media department at Eagleview.
Here is the software we use for each class. If the software is freely available, a link is also provided.

Whenever possible, I choose software that is free (and Open Source) for students to use at home.

Photography:

  • Adobe Photoshop

Film and Video:

  • Adobe Premiere Pro
  • Adobe After Effects

2D Animation:

  • OpenToonz
  • Stykx Stick Figure Animation
  • Anime Studio Pro (Now called Moho)
  • Krita

3D Animation:

  • Blender

Stop Motion Animation:

  • OpenToonz*
  • DragonFrame

Audio Editing:

  • Audacity

Video Game Design

  • Unity
  • GameMaker: Studio
  • LMMS - Music Creation Software

Broadcast:

  • Adobe Premiere Pro
  • TurtleText Teleprompter Software*

Other Useful Tools:

  • Greenshot ScreenShot Software
  • Open Broadcast Software - Screen Recording
  • Visual Studio Code - Text Editor
  • Inkscape - Vector Drawing

 

 

*I am a developer of this software.

In case you haven’t found the answer for your question please feel free to contact me .
